## 两个正序数组的中位数

题目：

```txt
给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的 中位数 。

算法的时间复杂度应该为 O(log (m+n)) 。

示例 1：

输入：nums1 = [1,3], nums2 = [2]
输出：2.00000
解释：合并数组 = [1,2,3] ，中位数 2
示例 2：

输入：nums1 = [1,2], nums2 = [3,4]
输出：2.50000
解释：合并数组 = [1,2,3,4] ，中位数 (2 + 3) / 2 = 2.5
提示：

nums1.length == m
nums2.length == n
0 <= m <= 1000
0 <= n <= 1000
1 <= m + n <= 2000
-106 <= nums1[i], nums2[i] <= 106
Related Topics
数组
二分查找
分治
```

解答：

| **维度**       | **暴力破解**        | **双指针**              | **极致优化 (二分)** |
| -------------- | ------------------- | ----------------------- | ------------------- |
| **时间复杂度** | O(N log N)          | **O(m+n)**              | O(log(m+n))         |
| **空间复杂度** | O(m+n) (开辟新数组) | **O(1)** (仅存几个变量) | O(1)                |
| **性能表现**   | 数据大时慢          | 速度快，内存极省        | 针对千万级数据最强  |

### 方案一、暴力破解

```java
 public double findMedianSortedArrays(int[] nums1, int[] nums2) {
        //合并数组
        int[] mergeArray = new int[nums1.length + nums2.length];
        System.arraycopy(nums1, 0, mergeArray, 0, nums1.length);
        System.arraycopy(nums2, 0, mergeArray, nums1.length, nums2.length);

        //数组排序
        Arrays.sort(mergeArray);

        //判断数组是奇数还是偶数
        int mid = mergeArray.length / 2;
        boolean isOdd = mergeArray.length % 2 == 1;
        if (isOdd) {
            return mergeArray[mid];
        } else {
            return (mergeArray[mid] + mergeArray[mid - 1]) / 2.0;
        }
    }
```

评估：

1. **时间复杂度**：O((m+n) log(m+n))。主要是 `Arrays.sort` 导致的，对于两个**已经有序**的数组，这种做法浪费了“有序”这个前提条件。

2. **空间复杂度**：O(m+n)。创建了一个全新的 `mergeArray`。



### 方案二、双指针合并（归并排序）

#### 逻辑优化点：

1. **不需要真的合并全部数组**：中位数只跟中间的一个或两个数有关。我们只需要找到中间位置就可以停止。
2. **利用有序特性**：比较 `nums1[i]` 和 `nums2[j]`，谁小就移动谁的指针。

#### 优化后的代码 (O(m+n) 方案)：

这是最容易理解且性能大幅提升的方案。

```java
public double findMedianSortedArrays(int[] nums1, int[] nums2) {
    int m = nums1.length;
    int n = nums2.length;
    int len = m + n;
    
    // 记录当前遍历到的值和上一个值（用于偶数情况求平均）
    int left = -1, right = -1;
    int p1 = 0, p2 = 0;
    
    // 只需遍历到总长度的一半即可停止
    for (int i = 0; i <= len / 2; i++) {
        left = right; // 暂存上一次的值
        
        // 核心逻辑：谁小移谁，注意处理数组越界
        if (p1 < m && (p2 >= n || nums1[p1] < nums2[p2])) {
            right = nums1[p1++];
        } else {
            right = nums2[p2++];
        }
    }

    // 判断奇偶返回结果
    if ((len % 2) == 0) {
        return (left + right) / 2.0;
    } else {
        return right;
    }
}
```



### 方案三、二分递归查找

其核心思想不再是遍历数组，而是**通过二分法不断排除掉不可能成为中位数的元素**。

### 1. 核心逻辑：转化为“寻找第 k 小的数”

无论中位数是奇数还是偶数，我们都可以将其统一为：

- **奇数：** 寻找第 (m+n)/2 + 1 小的数。
- **偶数：** 寻找第 (m+n)/2 和第 (m+n)/2 + 1 小的数的平均值。

因此，问题转化为：**给定两个有序数组，找到它们合并后的第 k 小的数。**



```java
public static double findMedianSortedArrays(int[] nums1, int[] nums2) {
        int totalLength = nums1.length + nums2.length;
        // 如果总长度是奇数，找第 (totalLength/2 + 1) 小的数
        // 如果总长度是偶数，找第 (totalLength/2) 和 (totalLength/2 + 1) 小的数的平均值
        if (totalLength % 2 == 1) {
            return getKth(nums1, 0, nums2, 0, totalLength / 2 + 1);
        } else {
            int left = getKth(nums1, 0, nums2, 0, totalLength / 2);
            int right = getKth(nums1, 0, nums2, 0, totalLength / 2 + 1);
            return (left + right) / 2.0;
        }
    }

    // 递归查找第 k 小的数
    private static int getKth(int[] nums1, int start1, int[] nums2, int start2, int k) {
        // 如果 nums1 耗尽，直接返回 nums2 中的第 k 个
        if (start1 >= nums1.length) return nums2[start2 + k - 1];
        // 如果 nums2 耗尽，直接返回 nums1 中的第 k 个
        if (start2 >= nums2.length) return nums1[start1 + k - 1];
        // 如果 k=1，返回两数组头元素的最小值
        if (k == 1) return Math.min(nums1[start1], nums2[start2]);

        // 每次排除 k/2 个元素
        int half = k / 2;
        int mid1 = (start1 + half - 1 < nums1.length) ? nums1[start1 + half - 1] : Integer.MAX_VALUE;
        int mid2 = (start2 + half - 1 < nums2.length) ? nums2[start2 + half - 1] : Integer.MAX_VALUE;

        if (mid1 < mid2) {
            // 说明 nums1 的前 half 个元素不可能是第 k 小的，排除掉
            return getKth(nums1, start1 + half, nums2, start2, k - half);
        } else {
            // 说明 nums2 的前 half 个元素不可能是第 k 小的，排除掉
            return getKth(nums1, start1, nums2, start2 + half, k - half);
        }
    }
```
