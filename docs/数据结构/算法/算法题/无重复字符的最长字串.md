## 题目

```txt
给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串 的长度。

示例 1:

输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
示例 2:

输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
示例 3:

输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
提示：

0 <= s.length <= 5 * 104
s 由英文字母、数字、符号和空格组成
Related Topics
哈希表
字符串
滑动窗口
```

## 解答

方案一：哈希表

```java
import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;
import java.util.Map;

class Solution {
    public int lengthOfLongestSubstring(String s) {
        if (s == null || s.length() > 500000) {
            return 0;
        }

        Map<Integer, Integer> noRepeatStrsMap = new HashMap<>(s.length());
        for (int i = 0; i < s.length(); i++) {

            List<Object> noRepeatStrs = new ArrayList<>();
            noRepeatStrs.add(s.charAt(i));
            for (int j = i + 1; j < s.length(); j++) {
                char c = s.charAt(j);
                if (noRepeatStrs.contains(c)) {
                    break;
                }
                noRepeatStrs.add(c);

            }
            noRepeatStrsMap.put(i, noRepeatStrs.size());
        }
        if (!noRepeatStrsMap.isEmpty()) {
            return noRepeatStrsMap.values().stream().max(Integer::compareTo).get();
        }
        return 0;
    }
}
```

总结：

### . 逻辑分析

解题思路是**暴力枚举**：

- 以字符串中的每一个字符作为起始点 `i`。
- 向后遍历 `j`，直到遇到重复字符，记录此时的长度。
- 最后从所有记录的长度中取最大值。

**逻辑上：** 这个方法是正确的，能得到正确结果。 **效率上：** 这个方法有巨大的优化空间。

#### 时间复杂度:

总时间复杂度在最坏情况下接近 $O(n^3)$。对于 LeetCode 上 5 * 10^5 长度的字符串，这个算法会**直接超时**，因为 n^2 的操作次数大约是 2500 亿次，远超 1 秒内的计算量上限（通常为 10^8）。

#### 空间复杂度：

在循环内部不断创建 `new ArrayList<>()`，且在外层维护了一个巨大的 `HashMap`，这会产生大量的垃圾回收（GC）压力，导致内存占用极高。



## 方案二，最优解，滑动窗口

核心逻辑：使用一个 `Map` 来记录字符上一次出现的位置。通过 `left` 和 `right` 两个指针动态维护一个窗口，遇到重复字符时，直接将 `left` 跳跃到重复字符的下一个位置

实现：

```java
 public static int lengthOfLongestSubstring(String s) {
        if (s == null || s.isEmpty()) return 0;

        // map 存储字符及其上一次出现的下标
        Map<Character, Integer> map = new HashMap<>();
        int maxLength = 0;
        int left = 0; // 滑动窗口左边界

        for (int right = 0; right < s.length(); right++) {
            char c = s.charAt(right);
            // 如果字符已存在，且在窗口内，更新左边界
            if (map.containsKey(c)) {
                left = Math.max(left, map.get(c) + 1);
            }
            map.put(c, right);
            maxLength = Math.max(maxLength, right - left + 1);
        }

        return maxLength;
    }
```
